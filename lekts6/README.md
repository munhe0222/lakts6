## Асуулт Backup гэж юу вэ?
##  Урьдчилсан нөхцөл
Нөөцлөлт хийхийн тулд таны компьютерт дараах зүйлс бэлэн байх шаардлагатай:
* MySQL Server суулгасан байх.
* Terminal эсвэл Command Prompt (CMD) дээр ажиллах эрхтэй байх.
* Өгөгдлийн санд нэвтрэх хэрэглэгчийн нэр болон нууц үг.

##  mysqldump ашиглан нөөцлөх
`mysqldump` нь MySQL-ийн хамгийн түгээмэл нөөцлөлтийн хэрэгсэл юм.

### 1. Ганц өгөгдлийн санг нөөцлөх
```bash
mysqldump -u [хэрэглэгчийн_нэр] -p [өгөгдлийн_сангийн_нэр] > backup_file.sql
Backup (нөөцлөлт) гэдэг нь өгөгдөл, файл эсвэл системийн мэдээллийг алдагдах, устах, гэмтэхээс хамгаалах зорилгоор тусад нь хуулбарлан хадгалах үйл явц юм.

Жишээлбэл, MySQL дээрх database-ийн backup авснаар сервер эвдэрсэн, алдаа гарсан, эсвэл өгөгдөл устсан үед өмнөх хадгалсан хуулбараас сэргээх боломжтой болдог.

Товчоор хэлбэл, backup нь **өгөгдлийн аюулгүй байдлыг хангах хамгаалалтын арга** юм.
#  MySQL Backup (Нөөцлөлт)

##  Backup гэж юу вэ?
Backup (нөөцлөлт) гэдэг нь өгөгдөл, файл эсвэл системийн мэдээллийг алдагдах, устах, гэмтэхээс хамгаалах зорилгоор тусад нь хуулбарлан хадгалах үйл явц юм.

---

## Яагаад backup хийх хэрэгтэй вэ?
- Өгөгдөл алдагдахаас хамгаална
- Систем эвдэрсэн үед сэргээх боломжтой
- Алдаа гарсан үед өмнөх төлөв рүү буцаана

---
